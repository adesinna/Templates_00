docker run:tag
    This will run the specific version in the tag, if the tag is not given
    it returns the latest tag.


docker run -it image_name:tag
    This will run the container in an interactive shell. Like for example you need a prompt, this will do it,
    note that it is quite different from exec which is to enter the container.


docker run -p host_port:docker_port image_name:tag
    This will run the container and map the container port to the specified host port


doker run -v path_on_host:path_in_container image_name:tag
    This will run persist the data.
